## Wave 2 (parallel, 8 lanes): editor and UI feature expansion

Every lane builds on the existing engine; each ships its tests and its mdbook section.

- Lane 2A, drawing and vertex editing: polygon and path drawing tools (click-to-place vertices, close/finish, width and endcap options for paths), vertex-level editing of existing shapes (drag vertex, insert and delete vertex on segment), rectangle-drag creation refined with modifier constraints (square, from-center). Undo-integrated like all edits.
- Lane 2B, boolean and transform operations in the UI: union, intersection, difference, XOR on the current selection (the `i_overlay` engine already exists; wire it to selection with layer rules and undo), offset/grow/shrink dialog, rotate and mirror controls with numeric entry, align and distribute (left/right/center/top/bottom, equal spacing).
- Lane 2C, productivity editing: copy, cut, paste, duplicate with offset repeat, an interactive array tool (rows, cols, pitch, live preview), move-by-delta numeric entry, and a via-stack builder dialog (pick two layers, places the cut plus enclosures per the technology rules).
- Lane 2D, snapping and guides: snap to geometry (vertices, edges, midpoints, centers) with visual snap indicators, user guides (drag from rulers), configurable grid and snap settings surfaced in UI.
- Lane 2E, layer and technology editing UI: a layer manager upgrade (reorder, recolor with a palette, per-layer fill styles, hide-others/solo), and a technology editor panel (dbu, layer definitions, editable rule values with validation) that round-trips the tech file.
- Lane 2F, search and selection depth: the query bar upgraded to a small filter language (layer, cell, area/width predicates, e.g. `layer:METAL1 width<400`), saved selection sets, select-similar, and an outline/hierarchy tree panel with click-to-locate.
- Lane 2G, view and export polish: light theme plus theme switching, per-document view bookmarks, export current view or selection to PNG and SVG from the UI, and a print-style monochrome render mode.
- Lane 2H, in-app agent UX upgrades (UI side only): conversation mode in the agent panel (follow-up messages refine the running session, e.g. "make those wires wider"), session history browser (list past transcripts, one-click replay), and an "ask agent to fix" button on any DRC violation that launches a scoped session with the violation region and rule as context.
