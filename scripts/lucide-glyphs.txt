# Lucide glyph subset for Reticle chrome (lane 1B, ADR 0097).
#
# One Lucide icon name per line (kebab-case, as in the Lucide codepoint map).
# Blank lines and lines starting with '#' are ignored. scripts/subset-fonts.ps1
# reads this file to (a) subset lucide.ttf to exactly these glyphs and (b)
# regenerate crates/reticle-app/src/theme/icons.rs (one SCREAMING_SNAKE_CASE
# `char` constant per name, sorted).
#
# Source of the list: docs/design/ia-inventory.md, walked top to bottom. Every
# reserved CommandId (section 4), every menu item (section 2), every Inspector
# group and section, the status bar, overlays, toasts, and the Layers panel gets
# a glyph. Extending the set is a one-line add plus a subset-fonts.ps1 rerun. We
# err on inclusion (ADR 0097): a few unused glyphs cost bytes, a missing one
# blocks a lane.
